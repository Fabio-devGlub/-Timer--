# ⏰ Relógio Digital

Projeto de **Relógio Digital** desenvolvido com **HTML, CSS e JavaScript**, que exibe a hora atual do sistema em tempo real diretamente no navegador.

O relógio atualiza automaticamente a cada segundo, mostrando **horas, minutos e segundos** de forma dinâmica.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## ⚙️ Funcionalidades

* ⏱️ Exibição da hora atual em tempo real
* 🔄 Atualização automática a cada segundo
* 🎨 Interface simples e moderna
* 📱 Estrutura leve e fácil de executar
